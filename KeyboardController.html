    KeyboardController

    This class initializes EventListeners that handles specific key presses. Keys Currently Bound: - enter: interact with tasks or addTask button - tab: loop through the list of tasks and the add task button - space: start/end the timer (restricted when editing text fields) - right arrow: mark current task as completed - +/=: expand/hide task list during work session
